# Curriculum Scope

Document Status: Draft  
Prepared On: 2026-06-28  
Project: Math Tutor Web Application

## 1. Purpose

This document defines the long-term mathematics curriculum scope for the platform. The goal is to eventually support learners from nursery-level early numeracy through advanced mathematics used by university students, PhD researchers, engineers, scientists, and NASA-level research users.

The scope is intentionally broad, but implementation must remain phased. The platform should not publish weak advanced content simply to look complete. Quality comes before coverage.

## 2. Curriculum Principle

Every topic should be teachable through a clear path:

1. Intuition.
2. Vocabulary.
3. Visual explanation.
4. Formal rule or theorem.
5. Worked examples.
6. Practice.
7. Common mistakes.
8. Applications.
9. Next steps.

Beginner lessons should use plain language. Advanced lessons should include formal notation, prerequisites, proofs where appropriate, and meaningful applications.

Nursery content is a special case. It should not assume reading ability. It should be delivered through audio prompts, visuals, simple interactions, and caregiver support.

## 5. Content Depth Standards

Each topic can have multiple depth levels:

- Concept page: Explains the idea simply.
- Lesson page: Teaches a specific skill.
- Practice page: Gives targeted questions.
- Proof page: Explains why a result is true.
- Application page: Shows real use cases.
- Research note: Introduces specialist or advanced context.

## 6. MVP Curriculum Boundary

The MVP should not attempt to cover all levels at full depth. The first release should focus on:

- Early numeracy seed pages.
- Arithmetic foundations.
- Fractions, decimals, and percentages.
- Algebra foundations.
- Geometry foundations.
- Basic statistics.
- Practice components.

Advanced and NASA-relevant topics should initially exist only as well-structured roadmap pages or seed pages until the foundation is strong.

## 7. Long-Term Content Rule

No topic should be marked complete unless it has:

- A clear learning objective.
- Original explanation.
- Correct notation.
- Worked examples.
- Practice questions.
- Answer explanations.
- Prerequisites.
- Related next lessons.
- Quality review.
