# Math Tutor

Math Tutor is a planned free mathematics tutoring web application. The long-term vision is to support learners from nursery-level early numeracy through advanced mathematics for university students, researchers, engineers, scientists, and aerospace users.

The first priority is the learning platform. AdSense monetization is a later phase after the site has useful original content, strong navigation, policy pages, and a child-safe approach for early learning content.

## Current Status

The project is in planning and foundation setup. The first runnable version is a dependency-free static MVP so it can be opened, tested, committed, and deployed without waiting on a broken local npm install.

## Product Direction

The site uses:

- A modern, elegant gray academic theme.
- Level-specific teaching language.
- Visual, audio, and touch-first learning for nursery learners.
- Clear written explanations for school learners.
- Formal notation, proofs, and applications for advanced learners.
