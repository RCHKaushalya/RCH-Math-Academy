Featured lessons

Study a short lesson

Counting objects

Count one object at a time. Say one number for each object and stop at the last object.

  1. Touch the first object and say one.
  2. Move to the next object and say two.
  3. The final number tells how many objects there are.
7 + 5 = 12

Adding numbers

Break a number apart to make ten, then add what remains.

  1. 7 needs 3 more to make 10.
  2. Split 5 into 3 and 2.
  3. 7 + 5 becomes 10 + 2 = 12.

Linear equations

A linear equation can be solved by keeping both sides balanced.

  1. Start with 2x + 3 = 11.
  2. Subtract 3 from both sides.
  3. Divide both sides by 2 to get x = 4.
