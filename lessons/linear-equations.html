Solving linear equations

A linear equation can be solved by keeping the two sides balanced. Whatever you do to one side, do the same thing to the other side.

Learning goal

Solve a two-step equation and check the answer.

The idea

The equal sign means both sides have the same value. To find the unknown, undo the operations around it in reverse order.

2x + 3 = 11

Worked example

Solve 2x + 3 = 11.

  1. Subtract 3 from both sides: 2x = 8.
  2. Divide both sides by 2: x = 4.
  3. Check: 2(4) + 3 = 8 + 3 = 11.

So x = 4.

Common mistake

Do not subtract from only one side. The equation stays balanced only when the same operation is done to both sides.

Practice

Question 1

Solve x + 5 = 13.

Answer: x = 8

Question 2

Solve 3x = 18.

Answer: x = 6

Question 3

Solve 4x - 2 = 10.

Answer: x = 3
