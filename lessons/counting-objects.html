Counting objects

Counting tells us how many objects are in a group. Count slowly, touch each object once, and stop when every object has a number.

Learning goal

Count a small group of objects and say the total number correctly.

The idea

Each object gets one number. Do not skip an object. Do not count the same object twice. The last number you say is the total.

1 2 3 4 5

Worked example

How many dots are shown?

  1. Touch the first dot and say one.
  2. Touch the second dot and say two.
  3. Continue until the last dot.
  4. The last number is four, so there are four dots.

Common mistake

Moving too fast can make you skip an object. Count slowly and point to each object while saying the number.

Practice

Question 1

How many dots are in the worked example?

Answer: 4

Question 2

If you count one, two, three, what is the total?

Answer: 3

Question 3

Should one object be counted two times?

Answer: No
